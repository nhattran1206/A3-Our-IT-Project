Skills and Jobs




1. Web developer

Responsibilities:

+ Website designing, building, or maintaining based on the company’s demand.
+ Conferring with teams to resolve conflicts, prioritize needs, develop content criteria, or choose solutions.
+ Maintain and control website update.
+ Change, update website contains based on the company’s direction.
+ Manage code to ensure it meets industry standards, and valid, website able to be compatible with other browsers, devices, system.


Technical skills:

+ Proficient in JavaScript, HTML, CSS and My SQL
+ Bachelor's degree in Web development or related field, or relevant experience
+ Knowledge of programming language and technical terminology


Soft skills:

+ Sufficient in group project, a true team player
+ Ability to turn thought into words, great verbal skills
+ English fluency is an advantage



2. Database developer

Responsibilities:

+ Collaborate with other domains, implement data solutions that solve business problems
+ Use specialized software to store and organize the website’s data
+ Data analytics


Technical skills:

+ Good understanding about structured query language (SQL)
+ General understanding of operating system (OS)
+ Knowledge of database concepts
+ Experience of JavaScript
+ Interested in data business analysis to build metrics and reports


Soft skills:

+ Experience of teamwork and group projects
+ Appropriate skill of collaboration, and problem solving
+ Ability to have autonomous learning and self-study



3. Software developer

Responsibilities:

+ Designing, developing and managing software programs
+ Testing and evaluating new programs
+ Troubleshooting and debugging programs
+ Maintaining and upgrading existing programs
+ Working with other developers to develop projects


Technical skills:

+ Have experience in developing and implementing software
+ Have knowledge of coding languages (C++, Java, JavaScript) )
+ Have experience in solving problems
+ Have mathematical aptitude
+ Have understanding about operating systems


Soft skills:

+ Able to perform teamwork well with other developers
+ Able to learn new languages and technology
+ Ability to pay attention to detail
+ Creativity is highly valuable
+ Process commercial and business awareness
+ Process commercial and business awareness



4. Security engineer

Responsibilities:

+ Developing Security System for the Web
+ Developing strategies to prevent any threat
+ Response to security threats, attack


Technical skills:

+ Certified Ethical Hacker (CEH)
+ Security Incident Handling and Response
+ Application Security Development
+ Advance Malware Prevention
+ Data Management Protection


Soft skills:

+ Good communication skill to effectively communicate with management and customer
+ Ability to think critically
+ Good analysis and solving problem skill
+ Ability to pay attention to every detail
